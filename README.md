# KityMinder-Electron

A desktop mind mapping application built with Electron, based on the KityMinder core.

## About (修改说明)
本项目是基于以下开源项目的修改版本，遵循 GPLv2 协议开源。

### Based on (基于项目)
* **[KityMinder Editor]** - [https://github.com/cloudlandboy/kityminder-editor](https://github.com/cloudlandboy/kityminder-editor) - GPLv2
* **[KityMinder-electron]** - GPLv2

## Quick Start
```bash
npm install
npm start
